## Installation

use to install E-Commerce Shop.


```git clone https://github.com/wondercrazy15/React_GraphQL_1 ```

1) Client side 
```bash
cd e-commerce-shop
npm install
npm start
```
2) Sever side 
```bash
cd server-app
npm install
node index.js
```

## Usage

1) Client side 
```python
npm i @material-ui/core
npm i @material-ui/icons
npm i apollo-boost
npm i axios
npm i base64-img
npm i graphql
npm i graphql-tag
npm i lodash
npm i material-ui-upload
npm i react-apollo
npm i react-axios
npm i react-dropzone-component
npm i react-redux
npm i react-responsive-carousel
npm i react-router-redux
npm i redux-thunk
   
```
2) Sever side 
